## 1 引言（Introduction）

神经元是高度极化的细胞，其树突与轴突（统称神经突，neurite）中的局部翻译对突触可塑性、
轴突导向与记忆形成至关重要。实现局部翻译的前提，是特定 mRNA 被定向运输并锚定到神经突，
而非滞留于胞体（soma）。经典工作表明，这种亚细胞定位在很大程度上由转录本序列编码——
尤其是 3'UTR 上的“邮政编码”（zipcode）元件与 RNA 结合蛋白（CPEB、FMRP、Staufen 等）
结合位点。然而，**究竟有多少定位信息可由序列预测、以及现代 RNA/DNA 基础模型
（foundation model）能否捕获这种信息，仍缺乏受控、可比的评估。**

近年 RNA/DNA 基础模型（RNA-FM、mRNA-FM、DNABERT-2、UTR-BERT 等）在多种序列任务上
表现亮眼，但针对 mRNA 亚细胞定位的评测多集中于人类细胞系的核/质/内质网等隔室
（APEX-seq、CeFra-seq），与**神经元 soma–neurite 定位**这一体系不同，物种也不一致。
更关键的是，现有评测常缺乏**泄漏安全（leakage-safe）**的划分：同源基因或相同序列若
跨训练/测试集出现，会高估性能。

本文做出如下贡献：

1. **构建首个泄漏安全、多来源的神经元 mRNA soma-vs-neurite 定位 benchmark**
   （小鼠 + 大鼠，18 个数据集，含 6 个 isoform 分辨率来源），并在**统一的、按基因/
   同源组/相同序列分组的冻结划分**下，跨转录本区域（3'UTR、CDS、全长）与分辨率
   （基因级、isoform 级）系统比较七类模型。
2. **发现基础模型的冻结嵌入并不显著优于简单 k-mer 基线**（唯一名义显著的例外不抗多重比较校正）。
3. **提出一个注意力门控融合模型**，将基础模型嵌入与可解释的序列特征融合，**在所有设置下
   显著且抗 Bonferroni 地超过 k-mer 及所有单一模型**；成分分解证明**基础模型是主要贡献者、
   且与转录本长度无关**，融合优于任一单一视图（协同效应）。
4. **生物学发现：定位信号在全长转录本上最强**（全长显著优于 3'UTR 或 CDS 单独），
   提示定位信息分布于整条转录本，而非局限于 3'UTR。

## 2 方法（Methods）

### 2.1 数据与标签规整
我们汇总了 18 个神经元 mRNA 定位数据集（小鼠、大鼠），统一为 **soma vs neurite 二分类**：
Cell_body 归为 soma；Dendrite / Neuropil / Axon / Neurite 归为 neurite；Synap 并入
Neuropil。ribosome-profiling 与生化分馏读出的 Ribosome / Cytoplasm 属**翻译/分馏轴**而非
解剖隔室，默认从定位任务中剔除（可选纳入并单独报告）。跨来源标签取并集；并采用
**来源可观测性掩码（source-mask）**：仅当某标签确由某来源测过时，其缺失才计为阴性，
避免将“未测”误判为“非定位”。人类样本仅 ~289，无法支撑有意义评估，故主分析限定
mouse + rat（“啮齿类 benchmark”）。

### 2.2 样本构建与区域规整
- **粒度**：基因级（每个 species-gene 取一条代表序列，canonical 优先、否则最长，标签并集）；
  isoform 级（保留每条不同的 3'UTR 序列，完全相同序列合并；多数 isoform 来源无 transcript_id，
  以序列本身作为异构体标识）。
- **区域规整**（`apply_region`）：依据 `sequence_type` 路由——仅显式标注为全长
  （cdna/transcript/mrna）的行用 Ensembl GTF 提取 3'UTR / CDS；已是目标区域的行
  （isoform 3'UTR、ALE 区间等）原样通过；**空 `sequence_type` 默认按全长处理并提取**，
  并可用 `--native-region-sources` 显式标注真正的原生区域来源，避免合并文件中
  全长 bulk 代表转录本被误当作 3'UTR。

### 2.3 泄漏安全划分
将样本按**基因、同源组（ortholog group）、相同序列（精确哈希，union-find 传递闭包）**
合并为不可拆分的分组单元；在分组层面做 70/15/15 的平衡划分（best-of-N 评分以兼顾各
标签的训练/验证/测试支持度），**划分一次冻结、供所有模型复用**。区域消融时，用
最严格的 CDS 基因集作为公共交集（CDS ⊆ 3'UTR ⊆ 全长），配合 `--restrict-to-split`
使三区域在**同一批基因、同一划分**上比较，唯一变量为区域。

### 2.4 模型族
所有模型共用同一数据/划分/标签/评估流程，唯一差异为编码器：
- **k-mer**：归一化重叠 k-mer 频率 + 逻辑回归（传统基线）。
- **定位专用网络**：RNATracker 式（CNN+BiLSTM+注意力池化）、DM3Loc 式（卷积 + 多头自
  注意力），以 one-hot 序列从零训练（为其紧凑复现，作受控基线）。
- **基础模型冻结嵌入**：UTR-BERT、RNA-FM、mRNA-FM、DNABERT-2，对长序列滑窗、token 均值
  池化得到嵌入，接逻辑回归头。codon 模型（mRNA-FM）仅用于 CDS（需阅读框）。
- **端到端微调**：在冻结嵌入之外，另对 RNA-FM 做整模型微调（含 truncate / sliding_mean /
  pooled_repr 三种长序列策略）。
- **提出的融合模型（fusion）**：将一个基础模型嵌入与**工程特征**作为两个视图，各自投影至
  共同空间，由一个**输入相关的门控**对视图做 softmax 加权融合，再经 MLP 头输出。其机制是
  让模型逐转录本决定信赖学习到的表示还是可解释特征。

### 2.5 工程特征
长度与 log 长度；GC 含量；16 维二核苷酸组成；以及一组定位相关 motif 密度（AU-rich
AUUUA、CPE UUUUAU、poly-U、GU-rich、G-quadruplex 代理 GGG、poly(A) 信号 AAUAAA），按
每 kb 归一化；二级结构 MFE 为可选项（依赖 ViennaRNA）。**长度是已知的混淆变量**，故另设
“仅长度”基线以量化其贡献。

### 2.6 训练目标与评估
统一采用**带掩码的二元交叉熵**：`pos_weight` 平衡类别、逐样本软置信度权重（soft 聚合）、
逐标签掩码（排除未观测标签）。阈值在验证集上选取，训练以验证集 macro-AUC 早停。
**模型选择仅用验证集**；测试集指标仅汇报一次。主比较指标为 **ROC-AUC（对类先验稳健，
便于跨设置比较）**，并辅以 AUPRC。所有模型间差异用**按基因/同源组的聚类自助法
（cluster bootstrap，2000 次重采样）**给出 95% 置信区间与双侧 p 值；对“提出模型 vs k-mer”
的四项比较施加 Bonferroni 校正。成分分解通过“仅长度 / 工程特征 / 仅基础模型 / 融合”四组
对照 + 两两自助法完成。

## 3 讨论（Discussion）

**基础模型单独并不优于 k-mer。** 在四个设置下，RNA/DNA 基础模型的冻结嵌入均未显著超过
k-mer（最佳者 ΔROC-AUC ≤ +0.018，唯一名义显著项不抗多重比较）。这与“更大的预训练模型必然
带来更好下游表示”的直觉相悖，但与该任务的客观难度一致（最优 ROC-AUC 约 0.65–0.70，
标签来自异质 assay、噪声较大）。对该领域而言，这是一个有价值的**警示性结论**：在缺乏强基线
对照时，基础模型的优势可能被高估。

**但基础模型携带互补信息，融合可解锁。** 我们提出的注意力门控融合模型在所有设置下
显著、抗校正地超过 k-mer 与所有单一模型；成分分解进一步表明，**基础模型在工程特征之上
仍贡献显著且与长度无关的增量**（全长 +0.085、isoform 3'UTR +0.029，均 p<0.001），且融合
优于任一单一视图。这说明基础模型学到的表示与“长度/组成/motif”等可解释特征**正交互补**，
单独看不出价值，融合后才显现。门控机制让模型按转录本自适应地分配两类信息的权重，是融合
奏效的关键。

**定位信号在全长转录本上最强。** 在同基因、同划分的区域消融中，全长显著优于 3'UTR 与 CDS，
而后两者无差异。这**修正了“定位信号主要在 3'UTR”的经典侧重**——至少在该预测任务上，信号
分布于整条转录本，可能反映 CDS（密码子/翻译相关）与 5'UTR 的协同贡献，或更长上下文有利于
模型整合。这也解释了为何融合模型在全长输入上表现最好。

**与既有方法的关系。** 经典定位预测器（RNATracker、DM3Loc）针对的是不同隔室体系；我们将其
紧凑复现后纳入受控对比，它们与基础模型相当，但同样未稳定超过 k-mer——融合模型才是真正的
进步点。

**局限性。** （i）仅 mouse + rat，人类样本过少；（ii）标签来自异质 assay，存在噪声，限制了
可达上限；（iii）融合使用**冻结**嵌入——我们测试的 RNA-FM 全长微调仅达 ROC 0.652，低于融合，
但更大规模或区域感知的微调可能进一步提升；（iv）isoform 判别的对立样本主要来自单一策展
来源（Ciolli），且该来源的 soma/neurite 对立约 90% 可由 3'UTR 长度解释，需在长度残差化下
单独评估；（v）头条为 soma–neurite 二分类，5 隔室多标签作为附属。

**未来方向。** 规模化微调；为区域感知的双 tokenization 模型（如 mRNABERT）编写
CDS 边界感知的适配器；纳入更多物种；对融合模型学到的 motif 做机制性解析；以及以“同基因
异构体定位判别 + 长度残差化”作为 isoform 主图。

---

## 4 结论（Conclusion）

在神经元 mRNA soma–neurite 定位上，RNA/DNA 基础模型的冻结嵌入**单独并不优于简单 k-mer
基线**；但它们携带与可解释序列特征互补的信息，**融合后可得到显著、稳健且为唯一抗多重比较
的最优预测器**（全长 ROC-AUC 0.70 / AUPRC 0.74），其增益主要来自基础模型而非转录本长度。
定位信号在**全长转录本**上最强。我们开放该 benchmark、泄漏安全划分与融合模型，以支持
神经元 mRNA 定位预测的可复现研究。
